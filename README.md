# Raptor-Backend

### Tech Stack:
- Typescript
- PostgreSQL (Database)
- Prisma (ORM)
- HonoJS (routing framework)
- Cloudflare Workers (deployment)
- Neon.Tech (Deploying PSQL DB)
- Zod (Input validation)
- JWT (Authentication & Authorization)

```
npm install
npm run dev
```

```
npm run deploy
```
